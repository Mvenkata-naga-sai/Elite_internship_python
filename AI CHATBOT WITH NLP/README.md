# NLP Chatbot using Python (NLTK)

## 📌 Project Title

NLP-Based Chatbot using Python and NLTK

---

## 📖 Description

This project is a simple chatbot developed using Python and the Natural Language Processing (NLP) library NLTK. The chatbot can understand user input, process it using tokenization, and respond with appropriate predefined answers.

The system demonstrates basic NLP techniques and provides an interactive command-line chatbot experience.

---

## 🎯 Objectives

* To build a chatbot using NLP techniques
* To understand tokenization using NLTK
* To simulate human-like conversation using rule-based logic

---

## 🛠️ Technologies Used

* Python
* NLTK (Natural Language Toolkit)

---

## ⚙️ Features

* Interactive chatbot interface
* Tokenization using NLTK
* Rule-based response system
* Easy to customize knowledge base
* Error-free execution without external dataset issues

## 🧠 Working Principle

* The chatbot takes user input
* Converts text into lowercase
* Applies tokenization using NLTK
* Matches input tokens with predefined intents
* Returns a suitable response
