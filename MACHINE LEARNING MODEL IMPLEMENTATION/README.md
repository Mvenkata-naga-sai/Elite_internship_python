# Spam Email Detection using Scikit-learn

## 📌 Project Title

Predictive Model for Spam Email Classification

---

## 📖 Description

This project builds a machine learning model using **Scikit-learn** to classify emails as **Spam** or **Not Spam (Ham)**.
It demonstrates the complete workflow of a predictive model.

## 🧠 Model Workflow

1. Import libraries
2. Create dataset
3. Split data into training and testing
4. Convert text using CountVectorizer
5. Train model using Multinomial Naive Bayes
6. Predict results
7. Evaluate using:

   * Accuracy Score
   * Classification Report
   * Confusion Matrix
